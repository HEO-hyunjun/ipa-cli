# examples

Copy-paste material for setting up a vault, kept neutral (no personal vault
content).

## What lives here

- `sample_profile/` — a working vault-local JS plugin sample: one extra search
  channel, one extra rule, and a tune workspace layout. Copy its `.ipa/`
  directory into a vault to try vault-local plugins end to end. See
  `sample_profile/README.md` for the install and smoke-test steps.
- `testset.example.json` — a sample tune testset showing the eval-set shape
  `ipa tune` consumes.

These are documentation fixtures — safe to copy from, and they never reference
real note titles or paths.
